## Newstome 

A personalized news digest app powered by the NewsAPI. Browse top headlines by topic, search for anything, and save articles to read later — all in a clean newspaper-style interface.

# Live Site
View on GitHub Pages ← replace with your deployed URL

# What does it do?

Browse by topic: Switch between Top Stories, Technology, Business, Science, Health, Sports, and Entertainment
Search: Type any keyword to find matching headlines from across the web
Save articles: Bookmark any article to your personal reading list (saved in your browser)
Newspaper aesthetic: Clean, editorial design that's easy to read

# How to use it

Get a free API key at newsapi.org/register
Open index.html and replace YOUR_API_KEY_HERE with your actual key
Run it on a local server (e.g. VS Code Live Server) — NewsAPI free tier requires localhost
Click any topic button to load headlines, or use the search bar


Note: Without an API key, the app runs in demo mode with sample articles so you can preview the UI.
